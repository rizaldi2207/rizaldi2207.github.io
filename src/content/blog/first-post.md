Astro adalah framework web *all-in-one* baru yang dirancang untuk kecepatan. Berbeda dengan framework lain, Astro merender HTML di server secara default dan menghapus JavaScript yang tidak perlu.

## Mengapa Astro?

1. **Cepat secara default**: Astro mengirimkan HTML murni ke browser tanpa beban JavaScript yang tidak perlu.
2. **Fleksibel**: Anda bisa menggunakan React, Vue, Svelte, atau bahkan tidak menggunakan framework UI sama sekali.
3. **Mudah dipelajari**: Jika Anda tahu HTML dan CSS, Anda sudah siap menggunakan Astro.

### Struktur Proyek

Struktur proyek Astro sangat intuitif dan mirip dengan proyek web tradisional, namun dengan kekuatan komponen modern.
