Web terus berkembang. Dari halaman statis sederhana hingga aplikasi satu halaman (SPA) yang kompleks, dan sekarang kembali ke render sisi server (SSR) dan *island architecture*.

### Island Architecture

Astro mempopulerkan konsep *Island Architecture*. Ini berarti Anda dapat memiliki "pulau" interaktivitas di tengah lautan HTML statis.

Ini adalah pendekatan yang mengubah permainan untuk kinerja web, terutama pada perangkat seluler.
